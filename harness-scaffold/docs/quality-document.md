## Grading Scale

- **A**: Build green, tests 95%+, 0 known defects, low tech debt
- **B**: Build green, tests 85%+, 0-1 minor defects, moderate tech debt

- **D**: Build failing or tests < 70%, multiple defects
- **F**: Not functional
